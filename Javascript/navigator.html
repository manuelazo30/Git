Tiene información sobre la página que tenemos cargada en el navegador: Propiedades:

Propiedad Descripción
appCodeName Devuelve el nombre del código del Navegador
appName Devuelve el nombre del Navegador
appVersion Devuelve la versión del Navegador
cookieEnabled Devuelve si el navegador tiene las cookies habilitadas o no
geolocation Devuelve un objeto Geolocation que puede ser usado para determinar la posición del usuario
language Devuelve el lenguaje del navegador
onLine Devuelve si el navegador tiene conexióna a internet
platform Devuelve la plataforma del navegador
product Devuelve el engine del navegador
userAgent Devuelve la cabecera del agente enviado por el navegador


Métodos Descripción
javaEnabled() Devuelve si el navegador tiene Javascript habilitado
