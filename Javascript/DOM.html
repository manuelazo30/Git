En DOM hay varios tipos de Nodos, a saber:

      Nodos Element: Representa un elemento HTML
      Nodos de Texto: Están contenidos en algún nodo element y representan el texto que se ve
      10 más (atributos de un elemento, comentarios HTML y más información relevante para la página
Cada tipo de Nodo tiene un número asociado (Node Types - Named Constants)
La mayor parte del trabajo se harán con los nodos tipos 1,2 y 3 que son:
1.- Elementos
2.- Atributos de los elementos
3.- Textos de los elementos
