Tiene información sobre la página que tenemos cargada en el navegador: Propiedades:

Propiedad Descripción
hash Asigna o devuelve la parte de un link (#) de una URL
host Asigna o devuelve el dominio y el puerto de una URL
hostname Asigna o devuelve el dominio de una URL
href Devuelve la URL completa
origin Devuelve el protocolo, dominio y puerto de una URL
pathname Asigna o devuelve el path de una URL (El archivo html que se está consultando y su ruta relativa)
port Asigna o devuelve el puerto de una URL
protocol Asigna o devuelve el protocolo de una URL
search Asigna o devuelve el querystring de una URL


Métodos Descripción
assign() Lee un nuevo documento
reload() Vuelve a cargar el documento actual
replace() Reemplaza documento actual con uno nuevo
